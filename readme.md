# paramint-slides

Reveal.js slides powered by npm dependencies (no copied reveal.js source code).

## Setup

```bash
npm install
```

## Run

```bash
npm run start
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- Main deck markup: `index.html`
- Reveal initialization: `main.js`
- Reveal assets come from `node_modules/reveal.js`
